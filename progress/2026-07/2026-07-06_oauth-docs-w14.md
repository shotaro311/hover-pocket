## 公式確認

- GitHub Pages はブランチ公開と GitHub Actions 公開をサポートし、ブランチ公開のフォルダーは `/` または `/docs` に限られるため、`site/` をそのまま使う本件では GitHub Actions を推奨にした。
- Google OAuth は external app、branding、authorized domains、Data Access の scopes、sensitive scope verification、Search Console 所有権確認の現行説明を公式ドキュメントで確認した。
- Google Calendar API の scope 一覧、`Events: insert`、`CalendarList: list` の認可スコープを確認し、`calendar.events` と `calendar.calendarlist.readonly` の現在の理由説明に反映した。
