Top-edge desktop utility panel

HoverPocket

HoverPocket opens a compact utility panel from the top edge of your desktop, so calendar work, clipboard history, sticky notes, timers, and quick calculations stay one hover away.

ホバーポケットは、画面上部へマウスを重ねるだけで、小さなユーティリティパネルを開けるデスクトップアプリです。 Calendar、Clipboard、Sticky Notes、Timer、Calculator、Mirror、Controls をひとつの場所から扱えます。

Features / 主な機能

Hover panel

Open the panel from the top edge, switch providers, and keep common tools close without a full app window.

Calendar

Connect Google Calendar, view events in the panel, and create, edit, or delete events on writable calendars.

Clipboard

Keep recent text and image clipboard history on your device, copy items back, drag them out, or clear the history.

Sticky Notes

Create local notes, edit them inline, reorder them, archive them, or delete them from the board.

Timer and Calculator

Run timers, pin timer presets, use Pomodoro-style sessions, and perform quick arithmetic inside the panel.

Mirror and Controls

Use a compact camera mirror and keep display, volume, and media controls close to the top edge.

Download / ダウンロード

Public builds are distributed from GitHub Releases. Use the latest release and choose the asset for your platform. Windows release assets use the HoverPocketWin name, and macOS release assets use the HoverPocket app package name.

公開ビルドは GitHub Releases から配布します。最新版のリリースを開き、利用するOS向けのアセットを選んでください。 更新確認も GitHub Releases を配信元として使います。

Privacy / プライバシー

HoverPocket stores Google refresh tokens in Windows Credential Manager and stores app data locally under %APPDATA%\HoverPocket. It does not run telemetry and does not sell or share Google Calendar data.

Google Calendar データは、接続時にパネル表示と予定操作のためだけに使います。詳しくは プライバシーポリシーを確認してください。
